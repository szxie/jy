Information Extraction on ECG Using OCR

Introduction

The goal of this project is to using OCR to extract useful Information from the ECG picture.

OCR is short for Optical Character Recognition, which is used to convert scanned or photographed images of typewritten or printed text into machine-encoded/computer-readable text.

At the same time, ECG pictures contain may useful information in a structured format, which makes it possible to correct errors in the OCRed result.

Data Set

The experimental data set for this project is 80 ECG pictures from Beijing University Hospital.

For these 80 pictures, roughly they can be divided in to 12 different types. The largest one contains 41 pictures and the experiment is focusing on these 41 pictures now.

The useful information in this pictures are all key-value-unit triples.

Approach

The whole process includes:

  1. Pre-processing on pictures.

    2. Thresholding technology is used to remove the noise in the picture.

  2. Using OCR engine.

    3. An open source OCR engine Tesseract is used.

  3. Post-processing on the OCR results.

    4. Since there are some position error in the OCRed text, the OCR results are re-organized by coordinate.

  4. Information extraction on the processed results.

    5. Triples extracted

    PR interval[148]ms
    QRS duration[88]ms
    QT/QTc[366/425]ms
    P-R-T axes[65, 67, 47]

Experimental Result

For all 41 pictures, which contain 5 useful key-value-unit triples, the roughly correction rate is 73% for now.
